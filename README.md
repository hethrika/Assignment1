# SaaS Review Scraper

## Objective
Scrape SaaS product reviews for a given company and date range and output them as JSON.

## Frontend
- `index.html` with inputs: company, start date, end date, source
- Click "Scrape Reviews" to see output in browser

## Backend
- Node.js server in `scraper.js`
- Runs on port 3000
- Saves output to `reviews.json`

## Installation
npm install
